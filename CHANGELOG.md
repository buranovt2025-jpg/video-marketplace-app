## [0.1.0] - 2024-12-22

### Добавлено
- **Initial commit:** Базовый TikTok Flutter клон загружен в репозиторий
- **Аутентификация пользователей** через Firebase Auth
  - Регистрация с email/паролем
  - Вход в систему
  - Выход из системы
- **Загрузка и просмотр видео**
  - Вертикальная прокрутка видео (TikTok-стиль)
  - Автовоспроизведение видео
  - Загрузка видео из галереи
  - Сохранение видео в Firebase Storage
- **Социальные функции**
  - Лайки к видео
  - Комментарии к видео
  - Подписки на пользователей
  - Отписки от пользователей
- **Поиск и профили**
  - Поиск пользователей по имени
  - Просмотр профилей пользователей
  - Статистика профиля (подписчики, подписки, лайки)
  - Список видео пользователя
- **UI/UX**
  - Адаптивный дизайн
  - Плавные анимации
  - Счётчики в реальном времени
- **Документация**
  - README.md с описанием проекта
  - CHANGELOG.md для отслеживания изменений
  - TODO.md с планом развития
  - TZ.md с техническим заданием

### Технологии
- Flutter SDK
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- GetX (State Management)
- MVC архитектура

### Известные ограничения
- Отсутствует функционал маркетплейса (в разработке)
- Нет системы оплаты
- Нет каталога товаров
- Нет функционала для продавцов
